# Premium Store

Landing page modern untuk katalog premium apps dengan order via WhatsApp.

## Stack
- React
- TypeScript
- Vite
- Tailwind CSS

## Fitur
- Neobrutalism UI
- Mobile-first responsive
- Search produk frontend
- WhatsApp CTA dengan pesan otomatis
- Toast sederhana
- Komponen reusable dan data-driven

## Struktur
Data produk dipisah dari komponen agar mudah dikembangkan ke dashboard admin, autentikasi, backend, dan database di masa depan.

## Jalankan lokal
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```
